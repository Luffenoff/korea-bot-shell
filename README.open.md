# Korea Bot — открытая оболочка

Публичная shell-версия без токенов, данных автомобилей и секретов.

## Быстрый старт

```bash
cp .env.example .env
cp cars.json.example cars.json
pip install -r requirements.txt
python bot.py
```

## Переменные окружения

См. `.env.example`
